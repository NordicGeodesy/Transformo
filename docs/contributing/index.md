# Contributing to Transformo

Transformo is an open source software package and welcomes contributions from
users. There are several ways to contribute to Transformo depending on skill set
and level. Of course code contributions make a big impact but so does reporting
bugs, suggesting new features and improving the documentation.

The software is in its early stages and contributions are more than welcome.
The project is managed on [GitHub](https://github.com/NordicGeodesy/Transformo).
Here you can [report a problem](https://github.com/NordicGeodesy/Transformo/issues),
[discuss an idea for a new feature](https://github.com/NordicGeodesy/Transformo/discussions)
or [create a pull request](https://github.com/NordicGeodesy/Transformo/pulls).
